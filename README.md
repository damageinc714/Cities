# Запуск
Для запуска лучше всего клонировать репозиторий в вашу среду разработки  
В таком случае вы не получите метку опасного файла(случай с Windows 11)  

Далее выставить выставить CitiesGame.WinForms как исполняемый и запустить  

На Windows 10 вы можете просто следовать такому методу:  
Для запуска программы необходимо скачать архив репозитория, распаковать. В папке найти файл с расширением .slnx  
Открыть его, выставить CitiesGame.WinForms как исполняемый в вашей среде разработке. Нажать кнопку запуска  

Windows 11 же ставит на скачанный архив блокировку, которую необходимо снять иначе будут возникать проблемы с запуском.  
<img width="1857" height="82" alt="image" src="https://github.com/user-attachments/assets/496ef2a3-5ad0-40c5-95d3-dc3332e462cf" />

Поскольку просто сняв блокировку с архива и распаковав на файлах она останется, нужно снять её через консоль:  
После скачивания архива, распакуйте его, зайдите в папку(например C:\Users\user\Downloads\Cities-master)  
В адресной строке пропишите 'powershell' и нажмите Enter.  
Введите следующую команду: 'Get-ChildItem -Recurse | Unblock-File'  
После этого можно открыть .slnx файл и привычно запустить его
