# Игра в города
Игровой проект на C#, созданный с помощью WinForms.
# Запуск
Для запуска лучше всего клонировать репозиторий в вашу среду разработки.  
В таком случае вы не получите метку опасного файла(случай с Windows 11).  

Далее нужно выставить CitiesGame.WinForms как исполняемый и запустить.  

На Windows 10 вы можете просто следовать такому методу:  
Скачать архив репозитория, распаковать. В папке найти файл с расширением .slnx  
Открыть его, выставить CitiesGame.WinForms как исполняемый в вашей среде разработке. Нажать кнопку запуска.  

Windows 11 же ставит на скачанный архив блокировку, которую необходимо снять иначе будут возникать проблемы с запуском.  
<img width="1857" height="82" alt="image" src="https://github.com/user-attachments/assets/496ef2a3-5ad0-40c5-95d3-dc3332e462cf" />

Поскольку просто сняв блокировку с архива и распаковав на файлах она останется, нужно снять её через консоль: 
После скачивания архива, распакуйте его, зайдите в папку(например C:\Users\user\Downloads\Cities-master).  
В адресной строке пропишите `powershell` и нажмите Enter.  
Введите следующую команду: `Get-ChildItem -Recurse | Unblock-File`.  
После этого можно открыть .slnx файл и привычно запустить его.  
# Правила
Для начала нужно выбрать режим:  
1. Против компьютера
2. С другими игроками(2-4)  

<img width="487" height="342" alt="image" src="https://github.com/user-attachments/assets/e5ee10c8-fae2-452f-9fa8-cd112bf445b4" />

Первый игрок должен вписать свой город, он отобразится в истории.  
Далее наступит следующий ход другого игрока. Ему нужно назвать город который начинается на последнюю букву предыдущего города.  
Если город заканчиается на буквы ь, ъ, ы вам нужно назвать город на букву которая идёт до них.  
Например: `Рязань` => `Норильск`  

<img width="487" height="378" alt="image" src="https://github.com/user-attachments/assets/4568eeda-b36a-4671-b344-2b6a864c9994" />  

У игрока всегда есть вариант сдатся, тогда он автоматически станет проигравшим, а остальные игроки выиграют.  

Также присутствует вариант прервать игру если появилась такая необходимость.  

При завершении игры, вам ничего не останется как только вернуться к меню выбора режима кнопкой "Назад".
